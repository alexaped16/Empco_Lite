Product Description

Barge Light

Part # 98-23400, 98-23400L, 98-23400LM, 98-23400M, 98-23400R, 98-23400RM

Commonly purchased with:

Pilot Location Light

Specification:

  • Tested and certified by US Coast Guard accepted independent laboratory.
  • Steady mode visibility exceeds 3 nautical miles.
  • Flashing light travels 2 nautical miles.
  • It comes standard with mounting magnets (70 lb. pull strength).
  • Steady mode will last 30 days using four 6-volt batteries.
  • Flash mode will last 120 days using four 6-volt batteries.
  • Steady on mode is available in red, green, and clear colors.
  • Flash mode is available in amber.
  • Available with optional Pilot Location Light attachment for the amber barge light.
